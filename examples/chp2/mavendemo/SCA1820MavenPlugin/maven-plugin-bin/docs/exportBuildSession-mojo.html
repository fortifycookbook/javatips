sca:exportBuildSession

Full name:

com.fortify.sca.plugins.maven:sca-maven-plugin:18.20:exportBuildSession

Description:

Exports a mobile build session for the specified build ID.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: pre-integration-test.

Required Parameters

Name Type Since Description
sourceanalyzer String - Location of the sourceanalyzer executable. Defaults to sourceanalyzer which runs the version on the PATH or fails if none exists.
Default value is: sourceanalyzer.
User property is: fortify.sca.sourceanalyzer.executable.
